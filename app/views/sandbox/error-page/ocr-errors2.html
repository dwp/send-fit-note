If you have been given a digital fit note

The fit note that was provided can't be processed by our system.

It could be because it is the wrong type of fit note.

Check that the Unique ID on the bottom left of your fit note says Med 3, as this is the only type of fit note we accept.

Where to find the code

For paper fit notes

Your file cannot be uploaded because:

The image is too light or too dark to read

You will need to take another photo. Make sure the fit note is well lit.

The image is not clear and readable

You will need to take another photo. Make sure the fit note is in focus and can be read.

The data matrix cannot be read

You will need to take another photo. Make sure the data matrix (the black and white square that looks like a QR code) is clear and in focus.
