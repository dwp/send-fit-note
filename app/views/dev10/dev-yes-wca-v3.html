Check if you need to send a fit note

If DWP has told you that you are in one of these groups:

  • the work-related activity group for ESA
  • the support group for ESA

you do not need to send a fit note unless your condition has changed.

If your condition has changed, you should tell DWP as soon as possible.

Talk to an advisor

  • Telephone: 0800 169 0310
  • Textphone: 0800 169 0314
  • Monday to Friday, 8am to 5pm (except public holidays)
